# Power of Two

- **Link problem:** [Power of Two](https://leetcode.com/problems/power-of-two/)
- **Difficulty:** Easy
- **Topic:** Math, Bit Manipulation, Recursion

## Problem Statement

## Example 1:

- **Input:** `n = 1`
- **Output:** `true`
- **Explanation:** `2^0 = 1`

## Example 2:

- **Input:** `n = 16`
- **Output:** `true`
- **Explanation:** `2^4 = 16`

## Example 3:

- **Input:** `n = 3`
- **Output:** `false`

## Constraints

- `-2^31 <= n <= 2^31 - 1`

## Follow up:
